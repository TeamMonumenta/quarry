Examples
========

The quarry source tree includes a few example uses of the ``quarry`` module:

.. code-block:: bash

    # List examples
    $ python -m examples

    # Run an example
    $ python -m examples.client_messenger

If you have ``quarry`` in your python search path, you can run the example
files directly.
